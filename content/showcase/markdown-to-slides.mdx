## 소개

기술 발표 준비 시 PowerPoint 대신 마크다운으로 내용을 작성하고,
CLI 한 줄로 Reveal.js 기반 슬라이드를 생성합니다.

## 사용법

```bash
npx md2slides presentation.md -o output/
```

## 주요 기능

- 마크다운 → HTML 슬라이드 자동 변환
- 코드 블록 구문 강조 지원
- 커스텀 테마 적용 가능
