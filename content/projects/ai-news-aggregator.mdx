## 프로젝트 소개

기술 블로그, 공식 문서, 뉴스 사이트에서 최신 기술 뉴스를 자동으로 수집하고,
Claude API를 활용해 한국어 요약을 제공하는 서비스입니다.

## 기술 구성

- Cloudflare Workers 기반 크론 수집기
- D1 데이터베이스로 기사 저장
- Claude API로 자동 요약 생성
- Next.js 프론트엔드로 목록·상세 조회

## 현재 상태

1주차 MVP 범위에서 기본 수집·요약 파이프라인을 구현 예정입니다.
